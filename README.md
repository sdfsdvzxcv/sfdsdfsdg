# Рильке - Маленький мостик к большой поэзии

Современное веб-приложение и телеграм бот для чтения и управления стихотворениями. Проект включает в себя полнофункциональный веб-сайт с админ-панелью и телеграм бота с рассылкой уведомлений.

## 🚀 Возможности

### Веб-приложение
- **Современный дизайн** с плавными анимациями и адаптивной версткой
- **Поиск и фильтрация** стихотворений по автору, жанру и периоду
- **Личная библиотека** для сохранения любимых стихов
- **Выделение цитат** в тексте стихотворений
- **Админ-панель** для управления контентом
- **Загрузка файлов** (Word, PDF, TXT) с автоматическим извлечением текста

### Телеграм бот
- **Случайные стихотворения** из библиотеки
- **Поиск по автору, стране и периоду**
- **Личная библиотека** пользователя
- **Регулярные уведомления** с настраиваемой частотой
- **Предложения новых стихотворений** для модерации

### Админ-панель
- **Статистика** проекта
- **Управление стихотворениями** и авторами
- **Рассылка** текстовых сообщений, фото, видео и GIF
- **Модерация предложений** пользователей
- **Загрузка контента** из файлов

## 📦 Установка

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd rilke-poetry
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения
```bash
cp .env.example .env
```

Отредактируйте файл `.env`:
```env
# Telegram Bot Tokens (получите у @BotFather)
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_BOT_TOKEN=your_admin_bot_token_here

# Admin IDs (ваш Telegram ID)
ADMIN_IDS=123456789

# Database
DATABASE_URL=poetry.db

# Flask Settings
FLASK_ENV=development
FLASK_DEBUG=True
```

### 5. Создание телеграм ботов

1. Откройте Telegram и найдите @BotFather
2. Создайте основного бота: `/newbot` → выберите имя → выберите username
3. Создайте админ-бота: `/newbot` → выберите имя → выберите username
4. Скопируйте токены в `.env` файл

### 6. Получение вашего Telegram ID

1. Напишите боту @userinfobot
2. Скопируйте ваш ID в переменную `ADMIN_IDS`

## 🚀 Запуск

### Запуск веб-приложения
```bash
python app.py
```
Веб-приложение будет доступно по адресу: http://localhost:5000

### Запуск телеграм бота
```bash
python bot.py
```

### Запуск админ-бота
```bash
python admin_bot.py
```

## 📱 Использование

### Веб-приложение

1. **Главная страница** - знакомство с проектом
2. **Стихотворения** - просмотр и поиск стихов
3. **Авторы** - список авторов и их произведения
4. **Библиотека** - личная коллекция сохраненных стихов
5. **Админ-панель** - управление контентом (логин: admin/admin123)

### Телеграм бот

Отправьте `/start` боту для начала работы:

- **📖 Получить стихотворение** - случайное стихотворение
- **🔍 Выбрать стихотворение** - поиск по автору/стране/периоду
- **📚 Личная библиотека** - ваши сохраненные стихи
- **➕ Добавить стихотворение** - предложить новое стихотворение

### Админ-бот

Отправьте `/admin` для доступа к админ-панели:

- **📊 Статистика** - обзор проекта
- **📝 Управление стихами** - добавление/редактирование стихов
- **👤 Управление авторами** - управление авторами
- **📨 Рассылка** - отправка сообщений всем пользователям
- **📋 Предложения** - модерация предложений пользователей

## 🗄 Структура базы данных

### Таблицы
- **users** - пользователи телеграм бота
- **authors** - авторы стихотворений
- **poems** - стихотворения
- **user_library** - личная библиотека пользователей
- **user_quotes** - избранные цитаты пользователей
- **user_submissions** - предложения от пользователей

## 🔧 API Endpoints

### Публичные API
- `GET /api/poems` - список всех стихотворений
- `GET /api/poems/random` - случайное стихотворение
- `GET /api/poems/<id>` - конкретное стихотворение
- `GET /api/authors` - список авторов
- `GET /api/authors/<id>/poems` - стихи автора
- `GET /api/stats` - статистика проекта

### Админ API
- `GET /api/admin/poems` - управление стихами
- `POST /api/admin/poems` - добавление стиха
- `GET /api/admin/authors` - управление авторами
- `POST /api/admin/authors` - добавление автора
- `GET /api/admin/submissions` - предложения пользователей
- `POST /api/admin/broadcast` - рассылка
