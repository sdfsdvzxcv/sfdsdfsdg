# 🎭 Рильке - Маленький мостик к большой поэзии

## 🚀 Быстрый запуск (5 минут)

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка ботов
1. Напишите @BotFather в Telegram
2. Создайте 2 бота: основной и админ
3. Скопируйте токены

### 3. Настройка переменных
```bash
cp .env.example .env
```
Отредактируйте `.env`:
```env
BOT_TOKEN=ваш_токен_основного_бота
ADMIN_BOT_TOKEN=ваш_токен_админ_бота
ADMIN_IDS=ваш_telegram_id
```

### 4. Запуск
```bash
python run.py
```

## 🎯 Основные функции

### Веб-приложение
- Поиск стихов по автору, жанру, периоду
- Личная библиотека с сохраненными стихами
- Выделение цитат в тексте
- Современный адаптивный дизайн

### Телеграм бот
- Случайные стихотворения
- Поиск по автору/стране/периоду
- Регулярные уведомления
- Предложения новых стихов

### Админ-панель
- Статистика проекта
- Управление стихами и авторами
- Рассылка всем пользователям
- Модерация предложений
