# 🎭 Проект "Рильке" - Полное резюме

## 📋 Что было создано

Полнофункциональная система для чтения и управления стихотворениями, включающая:

### 🌐 Веб-приложение
- **Современный дизайн** с плавными анимациями
- **Адаптивная верстка** для всех устройств
- **Поиск и фильтрация** стихотворений
- **Личная библиотека** пользователей
- **Выделение цитат** в тексте
- **Админ-панель** для управления контентом

### 🤖 Телеграм бот
- **Случайные стихотворения** из библиотеки
- **Поиск по автору, стране, периоду**
- **Личная библиотека** пользователя
- **Регулярные уведомления** с настраиваемой частотой
- **Предложения новых стихотворений**

### 👨‍💼 Админ-система
- **Статистика проекта** в реальном времени
- **Управление стихотворениями** и авторами
- **Рассылка** текстовых сообщений, фото, видео, GIF
- **Модерация предложений** пользователей
- **Загрузка контента** из файлов (Word, PDF, TXT)

## 🛠 Технологический стек

### Backend
- **Python 3.8+** - основной язык
- **aiogram 3.4.1** - для телеграм ботов
- **Flask** - веб-фреймворк
- **SQLite** - база данных
- **python-dotenv** - управление переменными

### Frontend
- **HTML5, CSS3, JavaScript** - веб-интерфейс
- **Font Awesome** - иконки
- **Google Fonts** - шрифты
- **Современные CSS анимации**

### Дополнительные библиотеки
- **PyPDF2** - работа с PDF файлами
- **python-docx** - работа с Word документами
- **Pillow** - обработка изображений

## 🎯 Ключевые функции

### Для пользователей
1. **Чтение стихотворений** - случайные и по выбору
2. **Поиск и фильтрация** - по автору, жанру, периоду
3. **Личная библиотека** - сохранение любимых стихов

### Для администраторов

2. **Статистика** - обзор активности пользователей
3. **Рассылка** - отправка сообщений всем пользователям
4. **Модерация** - одобрение предложений пользователей
5. **Загрузка файлов** - импорт стихов из документов

## 📊 База данных

### Таблицы
- **users** - пользователи телеграм бота
- **authors** - авторы стихотворений
- **poems** - стихотворения с метаданными
- **user_library** - личная библиотека пользователей
- **user_quotes** - избранные цитаты пользователей
- **user_submissions** - предложения от пользователей

### Связи
- Стихотворения связаны с авторами
- Пользователи могут сохранять стихи в библиотеку
- Предложения пользователей проходят модерацию
