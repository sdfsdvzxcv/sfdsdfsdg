# 🚀 Быстрый запуск проекта "Рильке"

## Шаг 1: Установка зависимостей
```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка переменных окружения
```bash
cp .env.example .env
```

Отредактируйте `.env` файл:
```env
BOT_TOKEN=ваш_токен_основного_бота
ADMIN_BOT_TOKEN=ваш_токен_админ_бота
ADMIN_IDS=ваш_telegram_id
```

## Шаг 3: Создание телеграм ботов

1. Напишите @BotFather в Telegram
2. Создайте основного бота: `/newbot` → имя → username
3. Создайте админ-бота: `/newbot` → имя → username
4. Скопируйте токены в `.env`

## Шаг 4: Получение вашего Telegram ID

1. Напишите @userinfobot
2. Скопируйте ваш ID в `ADMIN_IDS`

## Шаг 5: Запуск

### Вариант 1: Запуск всех компонентов сразу
```bash
python run.py
```

### Вариант 2: Запуск по отдельности

**Веб-приложение:**
```bash
python app.py
```
Откройте: http://localhost:5000

**Основной бот:**
```bash
python bot.py
```

**Админ-бот:**
```bash
python admin_bot.py
```

## 🎯 Что получится

- **Веб-сайт** с современным дизайном
- **Телеграм бот** для чтения стихов
- **Админ-панель** для управления контентом
- **Рассылка** уведомлений пользователям
- **Загрузка файлов** (Word, PDF, TXT)

## 📱 Тестирование

1. Откройте http://localhost:5000
2. Найдите вашего бота в Telegram
3. Отправьте `/start`
4. Попробуйте админ-панель (логин: admin/admin123)

## 🆘 Если что-то не работает

1. Проверьте, что все зависимости установлены
2. Убедитесь, что токены ботов правильные
3. Проверьте, что ваш Telegram ID указан верно
4. Посмотрите логи в консоли
